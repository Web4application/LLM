NAME

lmlm — Local Multi-Language AI Model CLI

SYNOPSIS

lmlm [command] [options]

DESCRIPTION

LMLM is a local multi-language AI toolkit. Use this CLI to generate, analyze, and manage code across multiple languages directly on your machine.

COMMANDS

Command Description
run Run the main LMLM AI workflow.
init <project_name> Initialize a new multi-language project with boilerplate structure.
prompt <text> Generate code or scripts from a natural language prompt.
analyze <path> Analyze scripts in the specified folder and output dependencies and execution order.
--help Display help for any command.

EXAMPLES

# Run the main workflow
lmlm run

# Initialize a new project
lmlm init my_project

# Generate a Python function
lmlm prompt "Create a Python function to reverse a string"

# Analyze an existing project
lmlm analyze ./examples/sample_project
